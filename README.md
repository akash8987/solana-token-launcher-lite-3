# Solana Token Launcher Lite

A streamlined tool designed for developers to interact with the Solana blockchain. This repository provides a clear, documented path to launching your own SPL token on Devnet or Mainnet.

## Features
* **Connection Management:** Seamlessly toggle between Devnet and Mainnet.
* **Token Creation:** Initialize a new Mint account with custom decimals.
* **Metadata Support:** Basic structure for attaching token information.
* **Minting Logic:** Simple function to mint tokens to a specific wallet.

## Quick Start
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Add your private key to `index.js`.
4. Run the script: `node index.js`.

## Security Note
Never share your private keys or commit them to version control. Use environment variables for production environments.
